관리자권한으로
euc-kr -> utf8 로 바꾸기.

```bash

# echo \$LANG

ko_KR.EUC-KR

# vi /etc/sysconfig/i18n

```

```bash
LANG="ko_KR.EUC-KR" 아래처럼변경
LANG="ko_KR.utf8"
```

```bash

# source /etc/sysconfig/i18n

```

EUC-KR 로 생성되있던 파일 utf8로 바꾸기.

```bash

# iconv -f=EUC-KR -t=utf8 파일명

```
